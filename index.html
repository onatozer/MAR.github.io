MAR: Multi-Agent Reflexion Improves Reasoning Abilities in LLMs

Onat Ozer1, Grace Wu1, Yuchen Wang1, Daniel Dosti1, Honghao Zhang1, Vivi De La Rue1
Machine Learning Research FA 25
1University of Michigan
Paper Supplementary Code arXiv

LLMs have shown the capacity to improve their performance on reasoning tasks through reflecting on their mistakes, and acting with these reflections in mind. However, continual reflections of the same LLM onto itself exhibit degeneration of thought, where the LLM continues to repeat the same errors again and again even with the knowledge that its wrong. To address this problem, we instead use multi-agen multi-persona debate between LLMs to generate the reflections. With this framework, we demonstrate increased performance over the baseline Reflexion in both question-answering and programming domains, most notably achieving a pass@1 accuracy of 82.6 on the HumanEval dataset with GPT 3.5, a higher accuracy than what GPT 4 achieves!

Under the traditional Reflexion framework, the agent uses a single LLM instance to reflect on its mistakes, and then uses these reflections as it attempts the task again. Under our proposed MAR framework, a multi-agent debate is initiated to generate the reflection, with the judge LLM giving the verdict.

Here we can see an example trial under the MAR framework, where each of the different personas generate a possible reason for failure, which the judge LLM then converts into a verdict that the LLM uses for self-reflecion.

As we can see, the MAR framework both outperforms the reflexion framework as well as the baseline gpt3.5 on question answering (HotPotQA) and programming (HumanEval).

MAR reflexion leads to consistent improvements in HotPotQA question-aswering, eventually outperforming the original Reflexion framework.
